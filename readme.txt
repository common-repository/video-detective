Plugin for showing trailers from Video Detective.

== Description ==

Plugin for showing trailers from Video Detective. Simply add [trailer:#published_id] to your post (where #published_id is the number for the trailer).

== Installation ==

To install the plugin manually:

1. Extract the contents of the archive (zip file)
2. Upload the video-detective folder to your '/wp-content/plugins' folder
3. Activate the plugin through the Plugins section in your WordPress admin
4. Update setting to your customer id and player id
